04

connector policy: fewer tools, better work.

Install the connector only when a workflow needs it. Then prune the tools.

  • Composio: OAuth layer for GTM SaaS. Prefer scoped actions over browser automation.
  • Filesystem: this repo and approved output folders only.
  • HubSpot: search, read, draft updates. No bulk writes.
  • Gmail: create drafts. No send until approved.
  • Slack / Drive / Sheets: selected resources, not whole workspaces.
