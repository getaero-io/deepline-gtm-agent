## Workflow

1. Start a Deepline Session UI plan before provider/search/enrichment work.
2. Search Deepline tools for the right provider path.
3. Run the smallest useful pilot before broader enrichment.
4. Build the account brief.
5. Inspect Deepline outputs/logs/usage before declaring completion.

## Output Pattern

- company snapshot
- likely GTM motion
- relevant signals
- suspected pain
- Deepline angle
- first-message angle
- open questions
- sources / proof status
